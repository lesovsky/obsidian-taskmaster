# Changelog

Все значимые изменения Obsidian TaskMaster документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
проект следует [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Компактный режим отображения карточек для работы с большим количеством задач
- Интернационализация (i18n) с поддержкой английского и русского языков

### Changed
- Обновлена структура данных настроек — добавлены поля `language` и `cardView`
- Версия схемы данных обновлена до `2` с автоматической миграцией

### Fixed
- Исправлено растягивание групп длинными названиями задач

## [0.1.0] - 2026-02-14

### Added
- Начальный релиз с визуальной доской задач и 6 группами
- Структурированные карточки задач в стиле SMART (Что, Зачем, Кто, Когда, Приоритет, Статус)
- Drag-and-drop перемещение задач между группами
- WIP-лимиты для групп с визуальной индикацией превышения
- Отмена удаления задач через toast-уведомления (7 секунд)
- Автоматические переходы статусов при перемещении между группами
- Автоочистка завершённых задач по настраиваемому сроку хранения
- Поддержка нескольких досок с переключением через селектор
- Индикация просрочки (красная рамка карточки при превышении дедлайна)
- Сворачиваемые группы (Backlog и Completed)
